# GreenGarden Lite

一个极简的移动端园艺科普微游戏，支持直接通过 GitHub Pages 访问手机端。

## 游戏说明

### 如何开始
1. 打开游戏后，从作物列表中选择一种作物（番茄/黄瓜）开始种植
2. 作物会自动进入生长循环，无需额外操作即可生长

### 基础操作
- **调节水分和光照**：拖动屏幕下方的滑块，将水分和光照保持在绿色最佳区间内
- **查看天气和挑战**：点击顶部的天气卡片和挑战卡片，可查看详细说明
- **应对突发事件**：当屏幕中央出现红色警告按钮时，点击按钮回答问题解决事件
- **查看图鉴**：点击右上角图鉴按钮，可查看已解锁的知识卡片

### 答题系统
- **阶段答题**：每完成一个生长阶段，会弹出知识卡片并进行答题，答对可获得永久加成
- **突发事件**：遇到病虫害、修剪、授粉等事件时，需要答题解决，答对可获得奖励
- **答题暂停**：答题期间游戏会自动暂停，关闭答题界面后继续生长

### 进阶技巧
- 保持水分和光照在最佳区间可触发连击，完成挑战任务可获得额外加成
- 关注节气变化和天气效果，及时调整水分光照策略
- 答对知识题可以解锁图鉴内容，并获得永久生长加速

## 部署说明
本项目为纯静态网页，无需构建。

## 目录结构
- `index.html`: 入口文件
- `data/`: 游戏数据 (作物、知识、节气)
- `*.js`: 游戏逻辑 (ES Modules)
- `assets/`: 图片资源
