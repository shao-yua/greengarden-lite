# 主题
农耕文化是中华文明的重要根基，园艺作为其中璀璨的分支，既承载着传统耕作智慧，又在现代农业发展中展现蓬勃活力。随着生物技术、智能装备等创新要素的融入，园艺产业正经历着从传统生产向高质量发展的深刻变革。为响应乡村振兴战略，落实“‘知农爱农、强农兴农’”育人目标，南京农业大学园艺学院拟举办“‘园’溯农耕，‘艺’启新质：趣谈园艺生产力密码”主题活动。本次活动将制作科普视屏形式，引导学生探索园艺生产的科技奥秘与文化内涵，帮助学生直观感受园艺产业中传统智慧与现代新质生产力的融合魅力，厚植“知农爱农、强农兴农”的责任情怀，营造崇尚科学、探索创新的校园氛围。

# 内容要求
围绕“园”溯农耕，“艺”启新质，可选三类主题：
（1）文化溯源：挖掘园艺领域的传统农耕文化内涵，可围绕“园艺农事技艺”（如传统嫁接手法、节气农耕安排、特色作物栽培口诀）、“园艺工具演变”（从手工农具到智能装备的迭代等方面展开；
（2）科技科普：解读现代新技术、新模式在园艺生产中的应用，重点呈现“新质生产力”如何赋能园艺产业；
（3）融合创新：呈现传统与现代结合案例（如嫁接技艺+分子育种）。

# 形式要求
作品须紧扣活动主题，运用新颖的网络媒介形式或数字技术手段创作，旨在趣味化、互动化地传播园艺知识与文化。作品应具有较强的网络传播属性和创新性。形式不限，鼓励创新。
例如：
  交互式作品：如H5页面、小程序、小游戏等。
  动态可视化作品：如数据可视化动画、动态信息图等。
  其他数字媒体形式：如VR/AR作品、数字沙画、Vlog等未在以上类别中涵盖的创新形式。
技术规格：因形式多样，不作统一要求。但作品须能够在主流网络平台或设备上顺畅运行与展示。如为线上作品，请提交访问链接及必要的操作说明；如为本地文件，请确保可在常见终端（如电脑、手机）上正常打开，文件大小原则上不超过500MB。
